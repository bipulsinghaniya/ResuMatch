# 🚀 ResuMatch — AI Resume vs Job Description Analyzer

> 💡 Built as a real-world project to help students optimize resumes for job applications.

ResuMatch is an AI-powered web application that helps users improve their resumes by comparing them with job descriptions and generating smart, actionable suggestions.

---

## ✨ Features

* 📌 Paste any Job Description
* 📄 Generate or preview Resume
* 🤖 AI-powered Resume Analysis
* 📊 Smart suggestions to improve resume
* 🎨 Multiple Resume Layouts (Left-Right / Top-Down)
* 📥 Download Resume as PDF
* ⚡ Modern and responsive UI

---

## 🧠 How It Works

1. Paste a Job Description
2. Add your resume details
3. AI analyzes:

   * Skills match
   * Missing keywords
   * Experience gaps
4. Get:

   * Improved Resume
   * Personalized suggestions

## ⚙️ Installation & Setup

### 1. Clone Repository

git clone https://github.com/bipulsinghaniya/ResuMatch.git
cd ResuMatch

---

### 2. Setup Backend

cd backend
npm install

Create `.env` file:

PORT=3000
GEMINI_API_KEY=your_api_key_here

Run backend:

nodemon server.js

---

### 3. Setup Frontend

cd frontend
npm install
npm start
